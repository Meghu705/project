# Project title
Personal Protfolio Website

# Project Description
This is a simple "personal portfolio website" created using "HTML5".  
The website showcases my basic information, skills, and contact details. 
A responsive and modern "CSS" design for a personal portfolio website that showcases profile, skills, and contact information.
It uses clean layouts, CSS variables, and media queries to ensure a professional look across all devices.
It demonstrates JavaScript concepts such as "DOM manipulation, event handling, form validation, dark mode, and local storage".
