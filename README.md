# E-commerce Product Landing Page – ShopNow

##  Project Overview
ShopNow is a responsive E-commerce Product Landing Page developed using
HTML, CSS, and JavaScript.  
The project showcases multiple products with dynamic price updates,
quantity selection, and add-to-cart functionality, providing a smooth
user experience.

##  Objectives
 To design a responsive e-commerce website
 To display multiple products dynamically
 To implement basic cart functionality using JavaScript
 To practice front-end web development concepts

##  Technologies Used
 "HTML5" – Page structure and content
 "CSS3" – Styling, layout, and responsiveness
 "JavaScript" – Interactivity and dynamic behavior

##  Pages Description

###  Home Page (`index.html`)
 Displays multiple products
 Quantity increase/decrease
 Dynamic price update
 Add to Cart functionality
 Responsive design

###  About Page (`about.html`)
 Information about ShopNow
 Company goals and features
 Full-screen responsive layout

###  Contact Page (`contact.html`)
 Contact form with name, email, and message
 JavaScript form submission alert
 Full-screen layout using Flexbox

##  JavaScript Features
 Dynamic quantity control
 Price calculation based on quantity
 Add-to-cart counter
 Multiple product handling using objects
 Basic form handling using JavaScript

##  Responsiveness
 Mobile-friendly layout
 Flexbox used for alignment
 Responsive navigation and sections

##  Testing
 Tested on different screen sizes
 Verified button functionality
 Checked form validation

##  How to Run the Project
1. Download or clone the project folder
2. Open `index.html` in any modern web browser
3. Navigate through Home, About, and Contact pages
